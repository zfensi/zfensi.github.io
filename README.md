# zfensi.github.io

最小 GitHub Pages 个人博客。

## 结构

- `index.md`：站点首页，直接写 Markdown。
- `_config.yml`：GitHub Pages / Jekyll 基础配置。
- `.github/workflows/pages.yml`：推送到 `main` 后自动部署。

## 使用方式

1. 修改 `index.md`。
2. 提交并推送到 `main`。
3. 等待 GitHub Actions 部署完成。

## 注意

如果你要使用根域名 `https://zfensi.github.io`，GitHub 仓库名必须叫 `zfensi.github.io`。
